# NetLP
Publically accessible proximity Wi-Fi finder. Find Wi-Fi networks and passwords to public networks near you. If the list is empty there are no available public networks near you. 

## Instructions
Required python packages: \
`pip install pillow`

Run: \
`python main.py`
